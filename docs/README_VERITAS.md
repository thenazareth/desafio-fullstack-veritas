INSTRUÇÕES PARA RODAR O PROJETO:

    PRÉ-REQUISITOS:
        - Node.js
        - npm
        - Go
        - executar comando: npm install create-react-app (instalar pacote react)

- Backend:
    Navegue até a pasta backend
    Execute o comando 'go run main.go'
    Para visualizar o backend: http://localhost:8080/tasks

- Frontend:
    Navegue até a pasta frontend
    Execute o comando 'npm run start'
    Caso o frontend não abra sozinho: http://localhost:3000/
